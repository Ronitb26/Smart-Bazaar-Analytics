# Smart Bazaar Retail Performance Dashboard

A business intelligence project that analyzes sales data for a retail chain store across different regions, store sizes, and item categories. This project connects a backend MySQL database to an interactive Power BI dashboard.

## 📊 Project Screenshot
![Smart Bazaar Dashboard](Screenshot%202025-12-12%20005653.png)

## 🛠️ Tech Stack
* **Database:** MySQL Server
* **ETL / Data Cleaning:** Power Query
* **Data Visualization:** Power BI Desktop

## 📈 Key Metrics
* **Total Sales:** $997.16K
* **Average Sales:** $141.24
* **Total Items:** 7K
* **Maximum Sales:** $266.89

## 💡 Key Insights
* **Top Performing Region:** Tier 2 cities bring in the most revenue ($393.15K).
* **Top Performing Store Type:** Supermarket Type 1 is the main revenue driver, outperforming Grocery Stores and Supermarket Type 2.
